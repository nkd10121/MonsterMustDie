【タイトル】
MonsterMustDie!

【ジャンル】
3Dタワーディフェンスアクション

【ゲーム内容】
プレイヤーと罠を駆使して迫りくる敵からクリスタルを守るタワーディフェンスアクションゲームです。

【プレイ人数】
1人

【プレイ時間】
2分～

【開発環境】
DxLib、C++、Unity(ツールとして使用)、Effekseer、HLSL

【対応プラットフォーム】
Windows

【操作方法】
コントローラーのみ。キーボード未対応。
左スティック : 移動
右スティック : 視点移動
Aボタン	　　 : ダッシュ(※1)
Bボタン	　　 : ジャンプ/決定
L,Rボタン　　: 装備切り替え
ZRボタン　　 : 攻撃/罠設置(※2)
Startボタン  : ポーズ

ZLボタン　　 : 罠回収(※3)
Yボタン　　　: ウェーブスタート(※4)
Xボタン      : 罠回転

Escapeキー   : 強制終了

※1	前方向に移動中のみ
※2	クロスボウを装備中は攻撃、罠を装備中は設置
※3	すでに設置されている罠を見ているときのみ
※4	最初のウェーブ開始前のみ


【ゲームの流れ】
3ウェーブにわたって敵が攻めてきます。
ウェーブは準備、戦闘を1セットとして、交互に準備ウェーブ、戦闘ウェーブ、準備ウェーブ...と切り替わります。

プレイヤーは左下の数字分、画面下部のコストを消費して罠を設置できます。
左下の数字は、敵を倒す、もしくは戦闘ウェーブクリアで取得できます。

右上の数字がクリスタルのHPで、0になるとゲームオーバーになります。
3ウェーブの間クリスタルを守り切るとゲームクリアです。

クリスタルのHPが減る条件は
①敵がクリスタルに到達した時
②プレイヤーのHPが0になって復活した時
の2パターンです。
